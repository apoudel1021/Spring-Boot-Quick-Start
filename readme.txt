Spring Boot Quick Start makes use of basic functionalities of Spring MVC. Basic concepts such as annotations, GET, UPDATE, POST and DELETE, have been introduced. 
This code makes use of tutorials from Javabrains and might have a lot of similarities. Also the lesson on Introduction to Spring Boot have been compiled from Javabrain tutorials.


How to import and build project?
Clone the project
Import in Eclipse as maven project
Go to Project Properties > Java Compiler > JDK Compliance and adjust JDK Compliance accoriding to your JDK version.
Go to Project Properties > Java Build Path > Libraries and adjust JRE system library
Go to Run As > Maven clean > Maven Install
